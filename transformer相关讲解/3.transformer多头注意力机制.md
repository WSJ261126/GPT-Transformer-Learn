## 核心参数对照表

| 参数 | 值 | 含义 | 代码位置 |
|------|--------|------|----------|
| `batch_size` | 64 | 并行处理的序列数 | 第12行 |
| `block_size` | 256 | 序列长度 | 第13行 |
| `n_embd` | 384 | 嵌入维度 | 第14行 |
| `num_heads` | 8 | 注意力头数 | 第15行 |
| `head_size` | 48 | 每个头的维度 (384÷8) | 第16行 |
| `n_layers` | 6 | Block层数 | 第17行 |
| `vocab_size` | 4244 | 词汇表大小 | 运行时 |
| `dropout` | 0.2 | 丢弃率 | 第18行 |

## 注意力计算细节

```
以序列 T=4 为例:

输入: [红, 楼, 梦, ，]  (第0,1,2,3位)

1. Q @ K^T (相似度矩阵):
   
        K[0]   K[1]   K[2]   K[3]
     ┌────────────────────────────┐
Q[0] │ 0.9   │ 0.1  │ 0.2  │ 0.0  │  ← "红"主要关注自己
     ├───────┼──────┼──────┼──────┤
Q[1] │ 0.5   │ 0.8  │ 0.3  │ 0.1  │  ← "楼"关注"红"和自己
     ├───────┼──────┼──────┼──────┤
Q[2] │ 0.2   │ 0.4  │ 0.7  │ 0.2  │  ← "梦"关注"楼"和自己
     ├───────┼──────┼──────┼──────┤
Q[3] │ 0.1   │ 0.2  │ 0.3  │ 0.6  │  ← "，"关注前面所有
     └───────┴──────┴──────┴──────┘

2. 掩码 (因果掩码):
   
     ┌────────────────────────────┐
     │ 0.9  │ -inf │ -inf │ -inf │  ← 第0位只能看自己
     ├──────┼──────┼──────┼──────┤
     │ 0.5  │ 0.8  │ -inf │ -inf │  ← 第1位只能看0,1
     ├──────┼──────┼──────┼──────┤
     │ 0.2  │ 0.4  │ 0.7  │ -inf │  ← 第2位只能看0,1,2
     ├──────┼──────┼──────┼──────┤
     │ 0.1  │ 0.2  │ 0.3  │ 0.6  │  ← 第3位看所有
     └──────┴──────┴──────┴──────┘

3. Softmax (归一化):
   
     ┌────────────────────────────┐
     │ 1.00  │ 0.00  │ 0.00  │ 0.00 │  ← "红"100%关注自己
     ├───────┼───────┼───────┼──────┤
     │ 0.37  │ 0.63  │ 0.00  │ 0.00 │  ← "楼"37%关注"红",63%自己
     ├───────┼───────┼───────┼──────┤
     │ 0.17  │ 0.28  │ 0.55  │ 0.00 │  ← "梦"关注"红""楼""自己"
     ├───────┼───────┼───────┼──────┤
     │ 0.11  │ 0.16  │ 0.21  │ 0.52 │  ← "，"关注所有
     └───────┴───────┴───────┴──────┘

4. 乘以 V 并求和 → 最终输出
```
