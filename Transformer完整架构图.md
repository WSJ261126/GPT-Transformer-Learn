## Block 内部结构详解 (Pre-Norm)

```
代码第152-168行:

输入 x (B, T, 384)
    │
    ▼
┌───────────────────────────┐
│ 1. LayerNorm (ln1)      │  代码: self.ln1 = nn.LayerNorm(n_embd)
│   Pre-Norm: 先归一化      │  作用: 稳定训练
└───────────┬───────────────┘
            ▼
┌───────────────────────────┐
│ 2. MultiHeadAttention   │  代码: self.sa = MutiHeadAttention(...)
│   多头注意力             │  作用: token之间交流信息
└───────────┬───────────────┘
            ▼
┌───────────────────────────┐
│ 3. + x (残差连接)        │  作用: 保留原始信息
│   x + sa(ln1(x))        │  解决梯度消失
└───────────┬───────────────┘
            ▼
┌───────────────────────────┐
│ 4. LayerNorm (ln2)       │  代码: self.ln2 = nn.LayerNorm(n_embd)
└───────────┬───────────────┘
            ▼
┌───────────────────────────┐
│ 5. FeedForward (前馈)    │  代码: self.ffwd = FeedForward(n_embd)
│   384 -> 1536 -> 384     │  作用: 逐token消化信息
│   扩大4倍再缩回          │  增强非线性
└───────────┬───────────────┘
            ▼
┌───────────────────────────┐
│ 6. + x (残差连接)        │  作用: 保留原始信息
│   x + ffwd(ln2(x))      │
└───────────┬───────────────┘
            ▼
      输出 (B, T, 384)
```

### Pre-Norm vs Post-Norm

```
Post-Norm (原始Transformer):
    x = LayerNorm(x + Sublayer(x))
    缺点: 深层时梯度不稳定

Pre-Norm (你的代码):
    x = x + Sublayer(LayerNorm(x))
    优点: 更稳定，训练更容易
```

## 各部分设计原因

| 部分 | 代码位置 | 设计原因 |
|------|----------|----------|
| Token Embedding | 第176行 | 把字符ID变成可学习的向量，理解字符含义 |
| Position Embedding | 第178行 | Transformer无序，注入位置信息 |
| MultiHeadAttention | 第123行 | 多头并行关注不同关系，信息更丰富 |
| Pre-Norm 结构 | 第166行 | 先Norm再计算，训练更稳定 |
| 残差连接 (+x) | 第166-167行 | 梯度直接回传，解决梯度消失问题 |
| LayerNorm | 第160-161行 | 稳定每层输入分布，加速训练 |
| FeedForward | 第138行 | 逐token非线性变换，增强表达能力 |
| 多级 Block x6 | 第181行 | 层层递进，提取更深层语义信息 |
| 最终 LayerNorm | 第186行 | 输出前最后一次归一化 |
| lm_head | 第189行 | 384维特征映射到4244维，预测字符 |

## 总结

**一句话理解Transformer：**

```
输入字符 -> 嵌入(词+位置) -> 6层Block(Pre-Norm+多头注意力+残差+前馈) -> 预测下一个字符
```

**每层Block做的事情：**

1. **Pre-Norm**: 先LayerNorm再计算，训练更稳定
2. **多头注意力**: 让每个token看到序列中所有token的信息
3. **残差连接**: 保留原始信息，解决梯度消失
4. **前馈网络**: 把信息消化成更强的特征
5. **层层堆叠**: 6层 = 6次信息交互+消化，提取深层语义
