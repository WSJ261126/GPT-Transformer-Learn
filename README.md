# GPT-Transformer 从零实现

一个简洁的 GPT（Decoder-only Transformer）实现，基于 PyTorch，适合学习 Transformer 原理。

## 项目简介

本项目参照了b站up主「黯淡蓝点的居民」的视频，基于《红楼梦》从零实现了一个字符级的 GPT 模型，帮助理解 Transformer 的核心机制：
- 多头注意力机制 (Multi-Head Attention)
- 位置编码 (Position Embedding)
- 残差连接 (Residual Connection)
- 前馈网络 (Feed Forward)

训练数据为《红楼梦》文本，用于演示文本生成任务。

### 说明
#### 参数方面
因为本项目的重点在于给大家展示transformer的核心机制，所以在参数配置上可能比较保守，仅用于演示。
若大家有兴趣，可根据自己的需求调整参数。
若想追求不错的训练效果，可以参照以下参数：
| 参数 | 建议值 |
|------|--------|
| batch_size | 64 |
| block_size | 512 |
| n_embd | 768 |
| num_heads | 12 |
| n_layers | 10 |
| learning_rate | 1e-4 |
| max_iters | 10000 |

若采用以上参数配置，可采用混合精度训练，以加速训练过程，如下：
                                                                                                                                                                                  
  训练循环改动（约5行）：                                                                                                                                                          
   混合精度训练在 main( ) 的训练循环里                                                                                                                                                         
 ``` 
  scaler = torch.amp.GradScaler('cuda')                                                                                                                                            
                                                                                                                                                                                   
  for i in range(max_iters):                                                                                                                                                       
      xb, yb = get_batch("train")                                                                                                                                                  
                                                                                                                                                                                   
      with torch.autocast(device_type='cuda', dtype=torch.bfloat16):                                                                                                               
          logits, loss = model(xb, yb)                                                                                                                                             
                                                                                                                                                                                   
      optimizer.zero_grad(set_to_none=True)                                                                                                                                        
      scaler.scale(loss).backward()                                                                                                                                                
      scaler.step(optimizer)                                                                                                                                                       
      scaler.update()    
   ```      
#### 学习率调度方面
学习率的设置对模型训练至关重要。在本项目中，我们采用了 3e-4 的学习率。若追求更好的效果，可根据实际情况调整学习率。因为固定学习率收敛效果差，cosine decay 是训练语言模型的标配，可以引入。

## 核心架构

```
输入: "今天天气"
   │
   ▼
[词嵌入 + 位置嵌入] → [Block × 6] → [输出]
   │                      │
   │                      ├─ 多头注意力 (8头)
   │                      ├─ Pre-Norm 结构
   │                      ├─ 残差连接
   │                      └─ 前馈网络
   ▼
输出: "很好" (预测下一个字)
```

## 训练建议

- 使用 GPU 可大幅加速训练
- 可根据需要调整 `max_iters` 和 `learning_rate`
- `block_size` 越大，模型能理解的上下文越长

## 扩展方向

1. **增加层数** - 修改 `n_layers`
2. **使用更大数据集** - 替换训练语料
3. **词级别模型** - 修改为词而非字符
4. **添加解码策略** - Beam Search、Temperature 等

## 关于 Transformer 讲解完整性

本项目覆盖了 Transformer 的核心机制，但对于全面理解仍有一些内容未涉及：

### 已覆盖 ✅

| 内容 |
|------|
| 词嵌入 + 位置嵌入 |
| Q、K、V 原理 |
| 单头/多头注意力 |
| 因果掩码 |
| 残差连接 |
| LayerNorm |
| FeedForward |
| Pre-Norm |

### 未覆盖内容 ⚠️

| 内容 | 说明 |
|------|------|
| Encoder（编码器） | 本项目为纯解码器，缺少Encoder |
| Encoder-Decoder Attention | 完整Transformer才有 |
| Sinusoidal位置编码 | 只实现了可学习方式 |

> **说明**：本项目是 **Decoder-only** 架构（GPT风格），适合学习文本生成任务。完整的Transformer（如原始论文）还包括Encoder部分，用于机器翻译等任务。

### 完整Transformer vs 本项目

```
完整Transformer (Encoder-Decoder):
┌──────────────┐     ┌──────────────┐
│   Encoder    │     │   Decoder    │
│  (双向注意力) │ ←──→ │  (因果注意力)  │
└──────┬───────┘     └──────┬───────┘
       │                     │
       │  编码器-解码器注意力
       └──────────┬──────────┘
                  ▼
              输出

本项目 (Decoder-only):
┌──────────────┐
│   Decoder    │
│  (因果注意力) │
└──────┬───────┘
       │
       ▼
    文本生成
```
